# Forage — Interactive Prototype

Forage is a camera-based recipe app. Point your phone at your ingredients and it suggests personalized recipes based on what's in your kitchen.

This prototype is your recurring design artefact throughout the course. You'll return to it in every session — first to understand the user and their context, then to critique specific design decisions, and finally to propose and implement real changes. There's no single right answer. Any change you can argue for is fair game — whether that's tweaking a line of text, reordering steps, removing screens, adding new ones, or rethinking the flow entirely.

---

## How to run it

1. Download or fork this repository
2. Open `index.html` in any browser (Chrome, Safari, Firefox)
3. No installation, no dependencies, no internet required

---

## Navigating the prototype

Click through the prototype as a real user would. For convenience, **← Prev** and **Next →** buttons above the phone frame let you move freely between screens at any time.

---

## Making edits

Open `index.html` in a code editor (VS Code recommended). All screens, styles, and interactions are contained in this single file. Save and refresh your browser to see changes.
